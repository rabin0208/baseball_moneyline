# Baseball Moneyline

ML project to predict which team wins a baseball game using pybaseball and the MLB Stats API.

## Setup

Create and activate the conda environment:

```bash
conda env create -f environment.yml
conda activate baseball_moneyline
```

## Environment

- **Python** 3.11
- **Data:** pandas, numpy, pybaseball, MLB-StatsAPI
- **Modeling:** scikit-learn
- **Exploration:** jupyter, matplotlib, seaborn
